# Media Storage Folder Structure

This folder contains all media files captured by the ArchaeoScan system.

## Folder Structure:
- recordings/ - Video recordings from camera stream
- snapshots/ - Screenshots/captures from camera stream  
- temp/ - Temporary files during recording process

## File Naming Convention:
- Videos: recording_YYYY-MM-DDTHH-MM-SS.webm
- Snapshots: snapshot_YYYY-MM-DDTHH-MM-SS.png

## Storage Management:
- Monitor folder sizes to prevent disk space issues
- Old files can be archived or deleted as needed
- Temp folder should be cleaned periodically
